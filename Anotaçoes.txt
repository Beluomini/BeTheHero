Métodos HTTP:

    GET     -> Buscar ou listar algo do back-end
    POST    -> Criar algo no back-end 
    PUT     -> Alterar info no banck-end
    DELETE  -> Deletar informação no back-end

Tipos de Parâmetros:

    Query Params: Enviados após o "?" feitos para filtros ou paginação
    Route Params: Identificar recursos como o id na URL
    Request Body: Usado pra alterar ou criar recursos
